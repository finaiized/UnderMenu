Hello!

This is a sample application built on some best-of-breed libraries such as jQuery, Backbone, Bootstrap, and Transparency. The aim is to make a visually-appealing application where users can customize and order plates of meals.

Dig in!

Undermenu is in development, but feel free to download the app and run index.html to take a look at the progress so far.
